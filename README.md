Program Description
It helped my team and me to export the base of SQL Store Procedure and Jaspersoft Report Layout from the excel file where previously Fund Accounting Department people made their Financial Report using Ms Excel.
So we dont have to manually design and type everything in by ourselves.

How it works
1. Open Terminal
2. Locate the folder of the program
3. streamlit run jasper_editor.py
4. Import .csv , xlsx, or .xls file that you want to export into the program.
5. Delete the unwanted rows and unwanted columns and press APPLY DELETION.
6. After that Renamed to the file name you want to export as
7. Press Download
